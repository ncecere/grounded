# Phase 6 - Robots & Policy Overrides

## Goals
- Provide enterprise controls for robots.txt compliance.
- Make overrides visible and auditable.

## Tasks
- Implement robots.txt enforcement in the fetch layer.
- Add per-source override flag with explicit logging when bypassed.
- Record robots decisions in crawl metadata.
- Add skip reasons for blocked URLs.

## Deliverables
- Robots enforcement plan and configuration spec.
- Logging and audit policy draft.
